# P2P ChatJS Express
JavaScript Node.js Express.js WebSocket P2P Chat Service with HTML and JavaScript Chat Client


## Project Structure
```shell
p2p-chatjs/
├── Dockerfile
├── docker-compose.yml
├── package.json
├── .env
├── certs/
│   ├── ca-cert.pem
│   ├── ca-key.pem
│   ├── server-cert.pem
│   ├── server-key.pem
├── node_modules/
├── public/
│   ├── index.html
│   ├── styles.css
│   ├── chat-script.mjs
├── src/
│   ├── server.mjs
│   ├── peersocket.mjs
│   └── peersocket_payload.mjs
└── README.md (optional)
```
